# BankBuddy AI Context Index

Files are exported in the order listed here by `basectl export-context`.

1. `README.md`
2. `PROJECT.md`
3. `ARCHITECTURE.md`
4. `COMMANDS.md`
5. `WORKFLOWS.md`
6. `DECISIONS.md`
7. `STATUS.md`

## How To Use This Pack

Read `PROJECT.md` first for the compact product summary. Use
`ARCHITECTURE.md` and `COMMANDS.md` when planning implementation work. Use
`WORKFLOWS.md`, `DECISIONS.md`, and `STATUS.md` before proposing changes,
opening issues, or deciding whether `.ai-context/` itself needs an update.
